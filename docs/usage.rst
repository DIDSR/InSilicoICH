Usage
=====

Installation
------------

.. code-block:: bash

   pip install InSilicoICH

CLI Usage
---------

The package provides CLI commands:

- ``generate``: Run simulation
- ``recruit``: Recruit patients

Example:

.. code-block:: bash

   generate --help

Python Usage
------------

You can also use the package in Python scripts:

.. code-block:: python

   import insilicoICH
   # ...
